I2207-08072 - Indisponibilidade Parcial - Hub Fundos

Ambiente:

Hub Fundos PROD

Sintoma:

Falha Funcioanl - cliente Modal

Causa Raiz:

Ao submeter um arquivo sem portabilidades para o serviço rtm-portability, a aplicação está retornando status OK(201), dando uma falsa impressão de que foram registradas portabilidades.

Solicitar ao usuário que não envie arquivos vazios.

Solução Definitiva:

Implementar tratamento para arquivos vazios no serviço rtm-portability.
